# **App Name**: Campus Pulse

## Core Features:

- Student Dashboard: Display dynamic insight cards for academics, well-being, finance, and sustainability, each with micro-interventions.
- Admin Dashboard: Display overview summary cards, system alerts, and an AI-generated insight summary.
- AI Insight Generation: Use the Gemini API to analyze aggregated student signals and generate a concise insight for administrators, acting as a decision-making tool.
- Data Storage: Store student signals, admin metrics, and alerts using Firestore.
- User Authentication: Secure user accounts for both students and admins to protect sensitive data.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6), evoking trust and intelligence, to be combined with the other design aspects of the existing Campus Pulse React dashboard.
- Background color: Light blue (#E3F2FD) provides a clean and unobtrusive backdrop, to be combined with the other design aspects of the existing Campus Pulse React dashboard.
- Accent color: A vibrant orange (#FFB74D) is used for interactive elements and calls to action, to be combined with the other design aspects of the existing Campus Pulse React dashboard.
- Body and headline font: 'Inter' (sans-serif) for clear, readable text in both dashboards. Note: currently only Google Fonts are supported.
- Maintain a card-based layout consistent with the existing Campus Pulse React dashboard to ensure intuitive navigation.
- Implement subtle transitions for micro-interventions and AI insights.
